# ColorfulWeather - 华美天气（已经上架酷安市场）
一款 Material Design 的渐变天气服务 App，包括天气信息显示、小时预报、一周预报、生活指数、城市选择与切换等功能. 引入 Bing 每日一图 Api 服务，每天都能更新图片.
- Apk 下载：https://www.coolapk.com/apk/255771
- 上架应用商店时间：2020/02/24
